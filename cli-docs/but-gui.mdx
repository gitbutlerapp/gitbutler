description: 'Open the GitButler GUI'
---

## Usage

```
but <PATH>
```

## Description

Opens the GitButler GUI application for the current project or a specified path.

## Arguments

### PATH

Optional path to open in the GitButler GUI. Can be:

- A directory containing a GitButler project
- A file within a GitButler project
- Any relative or absolute path

## Examples

Open GUI for current directory:

```
but .
```

Open GUI for a specific project directory:

```
but /path/to/my/project
```
