## Core colors

Core colors doesn't have any specific meaning or themes. They are just color ranges that used in the next color groups.
It's better to avoid using core colors directly in the app. Use color `Scale` colors instead.

Core colors still have some usage areas.
All colors in the app could be broken down to 4 areas â€” from the very light to the very dark:

- **Background colors** â€” colors that are used for backgrounds on large areas - like pages, modals, containers, etc.
- **Soft colors** â€” colors a bit darker and suitable for smaller elements like buttons, inputs, borders, etc.
- **Solid colors** â€” these colors are strong and dark. It's suitable small elements that should be highlighted - like error buttons, borders, alerts, etc.
- **Text colors** â€” these colors are dark and therefore suitable for text.

## Scale colors

Scale colors also doesn't have any specific meaning, but they have themes.

This is how you use scale colors:

```css
/* Use scale colors */
background-color: var(--clr-scale-ntrl-10);
color: var(--clr-scale-pop-90);
```
